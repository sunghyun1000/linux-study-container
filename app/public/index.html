🔑 SSH 접속 설정

SSH로 접속하려면 서버 내부에서 사전 설정이 필요합니다.

1. sshd 활성화
sudo systemctl enable ssh --now

2. 비밀번호 로그인 허용
Ubuntu는 /etc/ssh/sshd_config.d/ 안의 설정이 우선 적용됩니다.
해당 폴더 안의 파일을 열어 PasswordAuthentication yes 로 수정하고, 주석처리 되어있다면 해제하세요.
파일이 없다면 새로 ssh를 재시작 하세요.
sudo systemctl restart ssh

또는 공개키 로그인
클라이언트에서 키 생성:
ssh-keygen -t ed25519
공개키를 서버에 등록:
ssh-copy-id server@{외부IP}
또는 ~/.ssh/authorized_keys 에 직접 추가
